# ScalarDB & Spring Boot Coding Examples

コード生成スキル（`/generate-scalardb-code`）が参照する詳細な実装例を格納するディレクトリです。

## 作成済み

- `value-object-examples.md` - 値オブジェクトの詳細実装例（ID, 金額, 列挙型）

## 追加例の作成について

基本的なパターンは以下のコンパクト版Rulesファイルにインラインで記載済みです：
- `.claude/rules/scalardb-coding-patterns.md` — エンティティ、リポジトリ、トランザクション、設定、例外
- `.claude/rules/spring-boot-integration.md` — 依存関係、サービス、コントローラ、イベント、テスト、K8s

コード生成時に更に詳細な実装例が必要になった場合、このディレクトリにファイルを追加してください。
